## Log

| Log类方法 |  级别   |     作用     | 颜色 |
| :-------: | :-----: | :----------: | :--: |
|   Log.v   | Verbose | 显示全部信息 | 黑色 |
|   Log.d   |  Debug  | 显示调试信息 | 蓝色 |
|   Log.i   |  Info   | 显示一般信息 | 绿色 |
|   Log.w   | Warning | 显示警告信息 | 黄色 |
|   Log.e   |  Error  | 显示错误信息 | 红色 |
